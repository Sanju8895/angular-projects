Angular 19 Directive:

Note: directive is a class that adds behavior to elements in your Angular application. There are three main types of directives

  1. Component Directive
  2. Attribute Directive
  3. Structural Directive

Attibute Directive

NgClass: Adds and removes a set of classes

NgStyle: Adds and removes a set of HTML styles

NgModel: Adds two way data binding to an HTML form element
